# Requirements Agent Example Prompts

These prompts are used as golden examples for deriving `CircuitSpec` drafts.

1. `Make an ESP32 environmental sensor board with USB-C power and I2C sensors.`
2. `Create a small STM32 board with CAN and 12V to 5V power regulation.`
3. `Design a LiPo-powered BLE tracker board under 30x30mm.`
